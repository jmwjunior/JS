Esta é a parte de conteúdo, e toda parte de conteúdo estara sempre em HTML

A parte de estilo é configurado na parte do HEAD, colocando em os dois HEAD uma tag Style.Esta é a parte de estilo pra deixr tudo mais bonitinho.

Dentro da tag style vai pra uma nova tecnologia o CSS

Em CSS todo final de marcação em um ponto e virgula

Dentro de style criar uma tag body, que vai dizer pro programa que tudo que estiver dentro do body, vai receber as configurações que estamos programando pra ele.

Para alterar a cor de fundo como background-color e dar enter ja vai aparecer as principais cores, se desejar alterar pode só parar o mouse em cima que vai abrir um painel de cores.

O comando color dentro do body vai alterar a cor da letra

Terminada a parte de estilo que pode ser colocada em uma pasta separada. Vamos para a parte de interiração.
